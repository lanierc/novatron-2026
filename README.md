# Novatron Robotics 2026

Official repository for the Novatron Robotics 2026 competition robot code.

This project contains the software developed for the 2026 season.  
The codebase is structured to support modular development, testing, and competition deployment.

## About

Novatron Robotics is a student-led robotics team focused on engineering excellence, innovation, and competitive performance.

## Repository Purpose

- Robot control software
- Autonomous routines
- Driver control systems
- Testing and development builds

## Getting Started

Clone the repository:

```bash
git clone https://github.com/your-username/novatron-2026.git
```

Open the project in your preferred IDE and build according to your competition framework requirements.
